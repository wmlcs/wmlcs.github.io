# RWD自適應網頁設計
本網頁為滿足行動裝置使用者需求，使用RWD設計，主特色如下：
1. 導覽列（navbar）變成 縱向排列
2. Header 轉換為 上下堆疊
3. 「MENU」題示字樣會出現，提升使用者導覽清晰度
4. 大頭貼變為 150px，提高視覺清晰度
